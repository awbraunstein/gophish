This is a Golang client for the [Phish.net API](https://phishnet.api-docs.io/v3/the-phish-net-api/welcome).


# Supported API Endpoints

* /shows/query
* /setlists/latest
* /setlists/get
* /setlists/recent
* /setlists/tiph
* /setlist/random
